# Laporan Praktikum Modul 2 (Pengenalan Tools)

## Tujuan Praktikum
1. Dapat melakukan instalasi tool yang digunakan (Wireshark)
2. Dapat menggunakan tool (Wireshark) untuk menangkap dan mengidentifikasi paket data

## Langkah Percobaan
### 1. Mengunduh dan menginstal Wireshark dari website resmi

### 2. Menjalankan Wireshark

### 8. Keluar dari Wireshark

## Kesimpulan
Pada praktikum ini berhasil dilakukan instalasi dan penggunaan Wireshark untuk melakukan capture paket jaringan. Selain itu juga dapat mengamati paket HTTP yang ditangkap menggunakan fitur filter pada Wireshark
